 # **Список задач**

Проект реализует набор функций для работы со списком задач:

- addTask — добавляет задачу
- toggleTask — переключает статус done
- removeTask — удаляет задачу
- stats — возвращает статистику

 Каждая задача представлена объектом вида:
{ id: number, title: string, done: boolean }

Для запуска достаточно скопировать код в компилятор JavaScript и запустить программу
