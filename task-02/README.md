# **Использование CSS-переменных и тёмной темы**


Добавлена возможность переключение тем (светлая/тёмная) с сохранением в `localStorage`.
   - Используются CSS-переменные:
     --bg, --text, --card, --border
     - тема сохраняется в localStorage и восстанавливаться при обновлении страницы.

Запуск и проверка:
1. Откройте `index.html`. (после скачивания всей папки или репозитория)
2. Нажмите кнопку «Тема» (справа в шапке).
3. Убедитесь:
   - Цвета меняются: фон, текст, карточка.
4. Перезагрузите страницу, чтобы убедиться, что выбранная тема сохраняется.
